# North & Nest

A mobile-first GitHub Pages site for `northandnest/northandnest.github.io`.

## Included

- Public home page
- Local sign-in
- My Dashboard
- Daily weight, habit, mood, and note check-ins
- Automatic `localStorage` saving
- Auto-incrementing day count
- Yesterday Completed, Consistency, weekly summary, and AI Coach guidance

## Publish

Push these files to the root of `northandnest/northandnest.github.io`. GitHub Pages will serve `index.html` automatically.
